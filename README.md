# VibePick

VibePick is a 10-minute grocery delivery app built with Next.js and Lucide React.

## Features

* User authentication with Google and email/password
* User profile management
* Role-based access control
* Product catalog with filtering and sorting
* Product details page with reviews and ratings
* Shopping cart with real-time updates
* Checkout process with payment integration

## Tech Stack

* Next.js for server-side rendering and API routes
* Lucide React for UI components and styling
* MongoDB for database storage
* Mongoose for database modeling and schema validation
* Zod for schema validation and type checking
* React Hook Form for form validation and submission handling
* NEXT-AUTH for authentication and authorization

## Development

* `npm install` to install dependencies
* `npm run dev` to start development server
* `npm run build` to build production bundle
* `npm run start` to start production server

## Deployment

* `npm run deploy` to deploy to Vercel platform
